public class MyClass {
    public static void main(String args[]) {
      String s1 = "Hello ";
      s1+="World";
      
      String s2 = "Hello";
      s2+=" ";
      s2+="World";
      
      String s3 = s1.intern();
      String s4 = s2.intern();
      
      boolean b1 = s3==s4;
      
      boolean b2 = s3.equals(s4);
      
      

      System.out.println("output - " + b1 +" - "+b2);
    }
}

output - true - true

When you call intern() on a String object, Java checks if the string value is already present in the pool of interned strings. If it is, then it returns a reference to that string in the pool. If it is not present in the pool, it adds the string to the pool and returns a reference to the newly interned string.
